void app_letimer_pwm_open ( float  period,
float  act_period,
uint32_t  out0_route,
uint32_t  out1_route 
)
static

This function sets up the LETIMER to create a PWM waveform.

PWM specifications are stored in a struct (APP_LETIMER_PWM_TypeDef) which gets passed into the letimer_pwm_open function.

Note
PWM is set by default to be idle LOW. This function does not start the PWM waveform.
Parameters
[in]period,act_periodPeriod is the full period of the PWM waveform during both active and idle sections. Act_period is the time that the PWM waveform will be active (HIGH) for.
[in]out0_route,out1_routeout0_route, out1_route are the routes from LETIMER to the LED.
