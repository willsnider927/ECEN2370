◆ si1133_configure()

static void si1133_configure ( void  )
static

Configures the si1133 to be ready for force read.

sends and retrieves a series of packages through i2c to correctly configure the si1133 for use with it's white sensing diode. Confirms successful commands through checking the si1133's command counter.

Note
private function called within the file by the initialization function
