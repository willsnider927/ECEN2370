◆ Si1133_configure()

static void Si1133_configure ( void  )
static

Configures the Si1133.

Sets various parameters in the Si1133 to prepare for taking measurements. This includes selecting the white photodiode for measurements.

Note
If writing to registers is unsuccessful (command counter doesn't increase), program will enter EFM_ASSERT
